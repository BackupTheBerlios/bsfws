Running BWS applications is enabled by the BWS runtime environment, consisting of the \texttt{BWSApplet} applet and the \texttt{ScriptString} and \texttt{JSNode} 'helper' classes.

%When a BWS document has been rewriten to a BWS/HTML document, it can be loaded and executed in any BWS supporting web browser\footnote{Any web browser that supports Java, JavaScript/DOM and LiveConnect}. To achive this, the BWS runtime environment, consisting of a Java applet and two 'helper' classes, is contained in the document.
